# 规则

事件 -> 条件 -> 动作

当给定的事件发生，且指定的条件满足时，则触发规定动作的执行

## 手势

[weex 手势](https://weex.apache.org/zh/docs/events/gesture.html#%E6%89%8B%E5%8A%BF%E7%B1%BB%E5%9E%8B)

由手指或触控笔在触屏设备或者触摸板上触发

### touch identifier

表示一个触摸点在设备上的唯一标识

### touch events

1. start
2. end
3. move
4. cancel

### type

1. touch
2. press
3. pan
